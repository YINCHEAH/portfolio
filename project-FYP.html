FYP - Enhance Turbid Underwater Image

This is my final year project which is to enhance the turbid underwater image using technique multi-scale fusion and decomposing curve of attenuating color.

Aim: The aim of this project is To enhance the turbid underwater image to getting a clear underwater image.

Objectiv: The objective of this project is to using Multi-Scale Fusion to remove turbid underwater image. Besides that, enhance the Multi-Scale Fusion technique using Decomposing Curve of Attenuating Color.


In my FYP, I have research several techniques that can be enhance the turbid underwater image and I using technique Multi-Scale Fusion. The Multi-Scale Fusion is a technique to using two method (Equalization Histrogram with Gray World and Dark Channel Perior) to get the each feature. Then, using Multi-Scale Fusion to fusion the feature and get a clear underwater image. The benchmark result have show the Multi-Scale Fusion have getting a clear image. Furthemore, I have using the Decomposing Curve of Attenuating Color to replace Dark Channel Perior to getting more better quality clear image. The benchmark result have shown the Decomposing Curve of Attenuating Color have better result compare the Dark Channel Perior. The benchmark result after using the Multi-Scale Fusion also shown the new method (Equalization Histrogram with Gray World and Decomposing Curve of Attenuating Color) is better than the default method (Equalization Histrogram with Gray World and Dark Channel Perior).

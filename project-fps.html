Create A First Person Shooting Game

This is a mini project that create a FPS Game using three.js library. The challenges of this mini project is to do a collision of bullet with the target, the level changing when achieve the point, the movement of the avatar using mouse and keybroad, the target appear randomly to attract interesting of the game.


This is a group project, part involve is the contruct the screne, changing the level (score count) and a part of the collision.

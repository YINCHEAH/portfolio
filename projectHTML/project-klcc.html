construct a KLCC

The assignment aims is to contruct a computer generated image (CUI) of the KLCC using basic primitives. This assignment is using Python and the OpenGL to contruct it. The basic primitives like square, circle, triangle and polygon to combine it become a KLCC night view. There have a challenges which is need to understand the Catesian coordinate to place the shape to get desired graphics. Besides that, the knowledge of objec-oriented programming is help me to create the graphics become more flexible. Example like KLCC building and trees can simply to move, resize it.

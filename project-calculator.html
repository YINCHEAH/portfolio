Create a Calculator

This project is a school assignment that bulid a calculator. The challenges of the calculator is need to understand the step of calculation, the simbol place correctly to get expected results.


First version is a interatice Console-based calculator. There are simply to using Python to bulid a calculator. This calculator is using terminal to user input the value and to get the result. Besides that, there have help menu to let the user how to use it. Each user input have some description to let user input correctly.


Second version is to improve the console-based calculator become a Graphics User Interface(GUI) Calculator. In this version, there have using Tkinter as toolkit for pyhton to create a graphics interface.


Third version is to improve the GUI Calculator to advanced mathematical calculator. This version have adding more type of calculation, and also improve the interface of the calculator.

In this version is a group assignment, the involed part is Advanced Calculator, 2D and 3D vector, Conversion of Polar and Catesian coordinate.

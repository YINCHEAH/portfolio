FPS Shooting Game - HTML5, JavaScript

In this mini project, we have created an interactive and engaging FPS Shooting game. The main languages used in this game are HTML and JavaScript. The HTML file is used to live stream the game online by linking the JavaScript file created and the libraries used. Three.js is the main library used in this game for building up the scene, creating objects, setting up the camera, lighting, shadows, loader, and animation. The game created allows the user to shoot the target, and several levels are provided.

Apart from the graphical user interface, we provided users with a menu system with interactive buttons such as the start game, help, and exit button. For the help button, it will display a popup message box that contains instructions about “How to play” the game when the user clicks it. It also allows the user to click outside the box to close it. We also provided the user with a background video to make the menu interface more attractive.

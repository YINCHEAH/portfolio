The goal of this assignment is to create and develop a 2D interactive image editor which is known as a painting application. In this assignment, the GUI library which is Tkinter and the fundamental OpenCV is help us to create this painting editor.

In my painting editor, it provides various fundamental image editing features, which is including loading an image, opening an image, saving new image, and showing the image detail includes size of image. Other than that, my painting editor also have geometry image transformations which are scaling and rotation, color conversion form greyscale to HSV to RGB and vice versa. The cropping image, resize image and drawing various primitives which is line, ellipse, rectangle, and text also include in the painting editor.

This mini project is to produce an advanced image enhancement application that is capable to enhance the quality of hazy images. The methods used to enhance the quality of hazy images are Dark Channel Priority (DCP) and Contrast Limited Adaptive Histogram Equalization (CLAHE). In the image editor, many features that were included in the previous assignments were also employed, such as filtering, rotation, histogram equalization, and so on. The new interface shows 3 output windows simultaneously, the first window is to display the original image that the user has chosen, the second window displays the output, which is the image after editing, and the third window is to show the histogram and output of SSIM. Besides that, the editor is also able to compare the quality of the input and output images by using the Structural Similarity Index Method (SSIM) and Peak Signal to Noise Ratio (PSNR).

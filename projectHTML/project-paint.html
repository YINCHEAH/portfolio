Create a painting and image editor

The assignment is to build a painting and image editor application. This assignment is using Python and OpenCV to build the function of the painting and image editor application. The Tkinter is use to design the interface of the application. The challenges of this assignment is need to cearly understand the function and its output of OpenCV. Besides that, the painting and image editor application have many needed button, to create a nice interface will be a challenge.


In first version, this mostly is like painting application with some simple function to edit the image or drawing.


In second version, there have add more editing function to edit the image or drawing. The filter like Gaussian, Median, Average and bit-place slicing, the edge detection techniques like Canny, Prewit and Sobel have added in this version. Besides that, there have adjust the interface due to display the extra feature.


The below is a mini project which is an advanced image enhancement application. This application have remove all painting feature and added more methods to edit the image. The methods like Dark Channel Proirity (DCP) and Contrast Limited Adaptive Histogram Equalization (CLAHE). Besides that the Structural Similarity Index Method (SSIM) and Peak Signal to Noise Ratio (PSNR) added into this application to compare the quality of image before and after. In this application have add a real time camera to let the user to take a photo and to editing.

This is a group mini project, I involve the feature like Browse, Save, PSNR, SSIM, Histogram and real time camera.

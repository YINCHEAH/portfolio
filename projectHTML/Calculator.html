Calculator - Tkinter

In the calculator assignment, there have 3 assignment and will improve the calculator continuously. assignment 1 is interactive Console-based basic CG Calculator. assignment 2 is using tkinter to do a GUI-Tinter CG Calculator. assignment 3 is improve and expanding it into an advanced mathematical Calculator application

In the assignment 1, there is a interactive Console-based basic CG Calculator. It have three different type of calculation mode which is Basic Arithmetic, 2D Vector and 2X2 Matrix. Besides, calculator have provided “Help” to assist user to use this calculator. There also have “Exit” to end this calculator program.

In the assignment 2 is to construct and improve the source code from the prior interactive Console-based basic CG Calculator. All standard mathematical operations, such as addition, subtraction, multiplication, and division, are included in this calculator. This GUI-Tkinter CG Calculator also supports vectors (2D vectors) and operations like addition, multiplication, and subtraction. Addition, multiplication, and subtraction are examples of matrix operations on a 2x2 matrix. Only tkinter is used for the Graphics User Interface (GUI) in this second assignment.

In the assignment 3 is to improve our source code from the previous GUI-Tkinter Calculator by expanding it into an advanced mathematical Calculator application. This GUI-Tkinter Calculator includes all the standard mathematical operations, such as addition, subtraction, multiplication, and division as well as the Basic Trigonometry also include in this calculator.
